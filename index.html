host-with-benefits

Host files anonymously with an anonymous user. All users have an 8mb storage limit which cannot be exceeded. Each file uploaded will generate an unique code. These unique codes can be redeemed by any users to get access to the file.

You can create how many users you want (if you want to exploit the storage limits), but you will not be able to retireve lost users. If the username and/or password is forgotten, so are the files stored within that user. Look at your users as crypto wallets.

This project is made for fun as a litte proactive break from my studies, really.
